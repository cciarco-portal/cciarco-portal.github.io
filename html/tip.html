Tangent Background

Tangent Solutions, Inc. also called as Tangent, is an electronic payment service provider for acquiring banks in the Philippines. Tangent provides card present transactions processing payment gateway for card present transactions using EDC machines. We also provide payment gateway which routes the transactions to the acquiring banks. Tangent also provides merchants with POS devices to allow the merchants to accept payments using credit and/or debit card.
